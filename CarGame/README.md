# 極限登山車

*大二 - 遊戲程式設計專案*

## 使用說明

1. 開啟 Car.exe

2. 調整解析度和是否全螢幕 (建議不要全螢幕)

3. 開始遊戲

## 遊戲說明

1. 遊戲目標為駕駛車子登山並且成功至另外一端下山

2. 使用方向鍵朝各方向加速

3. 使用R可以回到紀錄點 (預設是起始位置)

3. 使用P可以更換紀錄點 (總共只能使用3次)

4. 按住F3可以使用前鏡頭
